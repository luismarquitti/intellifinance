# 02 - Architect/Tech Lead Agent

## Overview

The **Architect/Tech Lead Agent** performs technical planning and system design (Phase 2). Converts requirements from TPM/PO into actionable technical plans.

## Primary Responsibilities

- Design system architecture
- Make technology decisions
- Create implementation plans
- Define test strategy
- Assess technical risks
- Owner of `PLAN.md` and `CODE-STATE.md` files

## Key Deliverables

- Architecture decision records
- Implementation plan with phases and tasks
- Test strategy and approach
- Risk assessment with mitigations
- Updated `PLAN.md` with detailed roadmap
- Updated `CODE-STATE.md` after significant changes

## Tools Available

- `list_code_usages` - Code usage analysis
- `file_search` - Locate implementation patterns
- `semantic_search`, `grep_search` - Pattern discovery
- `read` - Code analysis

## Workflow

1. Read requirements from TPM/PO
2. Analyze existing architecture
3. Design solution approach
4. Break down into tasks
5. Estimate effort
6. Update PLAN.md
7. Present to user for approval
