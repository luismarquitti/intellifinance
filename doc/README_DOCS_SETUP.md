# Docusaurus Local Setup

This document provides instructions on how to set up and run the Docusaurus documentation site locally.

## Prerequisites

- Node.js (v18.x or higher)
- npm (or yarn/pnpm)

## Installation

1. Navigate to the `doc` directory:
   ```bash
   cd doc
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

## Running Locally

1. Start the development server:
   ```bash
   npm run start
   ```

   The site will be available at http://localhost:3000.

## Building

1. To create a production build, run:
   ```bash
   npm run build
   ```

   The build artifacts will be located in the `doc/build` directory.
